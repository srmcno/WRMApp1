You opened the source template

This file (wrm-app/index.html) is the development template — it only works when Vite is serving the project. To use the toolkit:

  1. Open OWRM-Toolkit.html at the top of this download instead. It is a single self-contained HTML file that works offline with no install — just double-click it.
  2. If you actually need to rebuild from source, run npm install && npm run build:dist inside the wrm-app/ folder. That regenerates the top-level OWRM-Toolkit.html.
